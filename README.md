## 关于项目

该项目处于初级学习阶段，还需要进行简单的手动配置即可食用。

### 食用教程

1. 安装运行环境

- [Python](https://www.python.org/)

2. 安装第三方库

``` shell
pip install -r requirements.txt
```

3. 修改配置

进入项目目录，找到 `config.ini` 文件，按照其中说明修改对于配置。

4. 运行脚本

修改项目主文件 `JdBuyer.py` 中 `sku_ids` 和 `area_id`。

然后运行程序：
``` python 
python JdBuyer.py
```
